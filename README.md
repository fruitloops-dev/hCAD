# 방그림

빈 자취방의 실측 치수로 평면도를 만들고 가구 배치를 미리 시험할 수 있는 모바일 친화적 웹앱입니다.

## 바로 사용하기

**[방그림 실행하기](https://fruitloops-dev.github.io/hCAD/)**

GitHub Pages에서 별도의 설치 없이 바로 사용할 수 있습니다. 처음 접속한 뒤에는 오프라인에서도 주요 기능을 사용할 수 있습니다.

## 사용 방법

1. 전체 도면의 가로·세로 크기를 입력합니다.
2. 방, 화장실, 주방, 현관 등의 구역을 추가합니다.
3. 구역의 크기와 위치를 조정하고 이어 붙여 실제 평면 모양을 만듭니다.
4. 가구를 추가한 뒤 드래그하거나 정확한 좌표를 입력해 배치합니다.
5. 완성한 도면을 PNG 이미지 또는 편집 가능한 JSON 파일로 저장합니다.

## 주요 기능

### 공간 구성

- 방·거실, 화장실, 주방, 현관, 발코니 등 구역 추가
- 구역 이동, 크기 및 위치 정밀 조정, 복제, 삭제
- 여러 직사각형 구역을 조합한 L자형·비정형 평면 구성
- 전체 도면 크기 및 20cm 단위 격자 맞춤 설정

### 가구 배치

- 침대, 책상, 옷장, 서랍장, 테이블, 소파 및 가전 빠른 추가
- 문, 창문, 기둥 등 구조물 배치
- 마우스와 터치로 이동하고 방향키로 1cm 단위 미세 조정
- 가구 크기와 벽에서의 거리 직접 입력
- 90도 회전, 복제 및 삭제
- 가구가 실제 구역 밖으로 나가면 경고 표시

### 저장 및 현장 사용

- 브라우저 자동 저장과 실행 취소·다시 실행
- 현장 메모와 실측 체크리스트
- PNG 이미지 내보내기
- JSON 도면 저장 및 다시 불러오기
- 모바일 반응형 UI와 오프라인 사용 지원

## 데이터 저장

작업 내용은 현재 브라우저에 자동 저장되며 외부 서버로 전송되지 않습니다. 다른 기기에서 이어서 작업하거나 백업하려면 JSON 도면 파일을 내려받아 보관하세요.

## 로컬 실행

개발하거나 수정 사항을 확인할 때는 저장소 폴더에서 다음 명령을 실행합니다.

```powershell
python -m http.server 4173
```

이후 브라우저에서 `http://localhost:4173`을 열면 됩니다.
