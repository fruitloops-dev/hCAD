# 방그림

자취방을 방문해 잰 치수로 대략적인 평면도를 만들고, 가구 배치를 미리 시험하는 모바일 친화적 웹앱입니다.

## 실행

별도 설치나 빌드가 필요 없는 정적 웹앱입니다. `index.html`을 직접 열거나 로컬 서버로 실행하세요.

```powershell
python -m http.server 4173
```

브라우저에서 `http://localhost:4173`을 열면 됩니다. 로컬 서버로 열면 서비스 워커가 등록되어 한 번 접속한 뒤에는 오프라인에서도 사용할 수 있습니다.

## 기능

- 방 가로·세로 치수 입력
- 침대, 책상, 옷장, 가전, 문, 창문, 기둥 등 빠른 추가
- 마우스와 터치로 이동, 90도 회전, 복제, 삭제
- 가구 크기와 벽에서의 거리 정밀 입력
- 격자 표시 및 20cm 단위 스냅
- 현장 메모와 실측 체크리스트
- 브라우저 자동 저장, 실행 취소/다시 실행
- PNG 이미지 및 편집 가능한 JSON 도면 내보내기
- 반응형 모바일 UI와 오프라인 사용 지원
